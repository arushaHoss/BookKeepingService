1. Add a .env file with the following informations - 
PORT= 3000
DATABASE_HOST=localhost
DATABASE_USER= #Add db user name here
DATABASE_PASSWORD= #Add db password here
DATABASE_NAME= #Add db name here
JSON_TOKEN_SECRET= mysecretkey
PASSWORD_HASH_SALT_ROUNDS=10
JWT_SECRET=mysecretkey
#All keys can be modified as per requirement

2. Type npm i in terminal to install all dependencies

3. Type npm run start:dev to start application
